
# Excel Dashboard

This folder contains the Excel dashboard developed for the Pizza Sales Analysis project.

The dashboard was built using Microsoft Excel to analyze pizza sales performance and identify key revenue trends, product popularity, and customer purchasing patterns.

## Excel Features Used

- Power Query – Used to import and clean data from multiple CSV files (orders, order_details, pizzas, and pizza_types).
- Power Pivot – Used to create relationships between tables and build a structured data model for analysis.
- DAX Measures – Created calculated metrics such as Total Revenue, Total Orders, Total Pizzas Sold, and Average Order Value.
- Pivot Tables – Used to summarize and analyze sales data across different dimensions.
- Pivot Charts – Used to visualize sales performance by pizza category, size, and time trends.

## Dashboard Highlights

- KPI indicators for key business metrics
- Sales performance analysis by pizza category and size
- Identification of top-performing and low-performing pizzas
- Time-based analysis of daily and monthly sales trends

The Excel dashboard enables interactive exploration of pizza sales data and helps identify key insights to support business decision-making.
